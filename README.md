# M25MAC010-A2

This repository contains implementations for:
Problem 1: Word Embeddings (CBOW &amp; Skip-gram) 
Problem 2: Character-Level Name Generation using RNN Variants

# How to Run the Project

## Clone Repository

```bash
git clone https://github.com/your-username/repo-name.git
cd repo-name
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Jupyter Notebooks

```bash
jupyter notebook
```

Open and run:

* `M25MAC010-Problem1.ipynb`
* `M25MAC010-Problem2.ipynb`

---

# Requirements

* Python 3.x
* PyTorch
* NumPy
* Matplotlib
* Scikit-learn
* Jupyter Notebook

---

# Key Learnings

* Understanding of word embedding techniques (CBOW vs Skip-gram)
* Implementation of sequence models from scratch
* Handling sequential data using RNNs and LSTMs
* Visualization of high-dimensional embeddings
* Generative modeling for text data

---

# Conclusion

* Skip-gram performs better for semantic understanding
* LSTM and BiLSTM significantly outperform Vanilla RNN
* Character-level models can generate realistic names with sufficient training
